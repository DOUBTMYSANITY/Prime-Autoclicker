# Unofficial Phasmophobia Reference

Offline reference pack for an unofficial Phasmophobia-style guide layout. Not affiliated with Kinetic Games.

This version keeps the useful parts of the original idea in a local, readable format: evidence lookup, ghost-specific tells, hunt thresholds, and quick equipment notes.

## Quick Use

1. Start with the three evidence matches.
2. If evidence is missing or inconsistent, fall back to hunt sanity, speed, and special tells.
3. Use behavior-based confirmation before locking in a ghost on 0-evidence rulesets.

## Evidence Legend

- EMF 5: event and interaction check.
- Ultraviolet: fingerprints, footprints, salt reactions.
- Ghost Orbs: camera-only evidence.
- Spirit Box: proximity voice check.
- Writing: ghost book interaction.
- Freezing: sub-zero breath or room temperature evidence.
- DOTS: projector-visible silhouette.

## High-Value Confirmation Notes

- If a ghost is hunting very fast and then collapses to a crawl near the player, check Deogen first.
- If the room is cold and the ghost speeds up in colder spaces, Hantu becomes a strong candidate.
- If a ghost feels quiet, avoid overcalling Shade without backup evidence.
- If the ghost throws multiple items in a burst, Poltergeist becomes the obvious suspect.
- If a camera shows orbs but the ghost behaves like something else, The Mimic is always worth checking.
- If the ghost seems to change over time rather than stay consistent, Thaye should stay on the list.

## Equipment Notes

- EMF reader: verify interaction hotspots and breaker activity.
- UV light: check fingerprints, footprints, and special salt reactions.
- Spirit box: test close-range responses in a quiet room.
- Ghost book: place it in the favorite room and wait for writing.
- DOTS projector: use camera angles when the room is not directly visible.
- Thermometer: watch for freezing evidence and room dips.
- Crucifix and incense: use them to catch hunt behavior and anti-hunt windows.

## Hunt Flow

1. Confirm the room.
2. Confirm at least one hard evidence type.
3. Test the ghost's speed and line-of-sight behavior.
4. Check whether the hunt threshold matches the expected sanity range.
5. Use special tells only after the core evidence is narrowed down.
